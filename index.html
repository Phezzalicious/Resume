  • CDKST Competency Wizard (2020)

    • Razor pages Application using EF core, Deployed on an Ubuntu 18.04 vps with MySql
    • The Code
    • The Deployed App
    Description

    CDKST (Competency, Disposition, Knowledge-Skill, Task) was a "Wizard" that would walk a user through creating a database record for an object the customer needed called a competency. This required the use of session variables to manage state throughout the application. On the last step, we would deserialize the session variable and use its contents to make calls to our database using a repository pattern. We would than take our results and build the competency object which is comprised of all of the other objects in the application and add it to the database as well. The code and application referenced above are my version of the project. I was not required to code one, but I needed to stay a step ahead of all four teams. It functions and does everything it is supposed to. The front-end was not a priority for our customer.
